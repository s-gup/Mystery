# Mystery

Multiplayer gaming app - solve crime ,mysteries by clues,chatroom discussions.

1.)	Game objective: 
A team of 3-4 people solve puzzles and mysteries (logical and fun) in a virtual chatroom. The team then works on the puzzle by working on the different hints given.
2.)	Game content: 
Puzzles and Mysteries (logical and fun) , Individual clues , Inbuilt chatroom discussions
3.)	Game assembly:
 Internet connectivity required 
4.)	Game setup: 
New users will have to register with a username and a minimum 6-digit password. Then the registered users will login with the same credentials.
5.)	Game Play: 
After logging in, user will have two options-
Create Room 
Join Room. 
•	Choose number of players (3 or 4) Choose theme (fun or logic) Acquire the Room Id  Share it with your friends.  
•	Your friends can join the Room by entering with the same Room Id. 
•	Room will be open for 3 minutes.
•	Each player of the team will receive separate hints for puzzles.
•	Analyse the hint in 3 minutes. Observe and analyse the hint carefully as it will be accessible only within that time frame of 3 minutes. 
•	You will be redirected to a chat screen where you can discuss hints with your friends and try to find the answer.  After 10 minutes, the leader (who created the room) will have the option to answer within 35 seconds.
6.)	Special Condition: 
•	After joining the Game Room, if the application goes in background state i.e. the app is minimized (like a user is on phone call etc.), the particular user will be kicked out of the Game Room and will not be able to continue the further quest with his team. 
•	If the specified number of players don't join the room within 3 minutes time frame, Room will be destroyed and you have to start from beginning.
7.)	Winning: 
After 10 minutes, leader (who created the room) will have the option to answer within 35 seconds. If he types the answer correctly the team will be the winner.
8.)	Gameplay variations:
 3 players or 4 players. 
9.)	Strategy tips: 
Most of the answers will be one word unless specified otherwise.
